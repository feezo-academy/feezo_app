Add Student
Enroll this student in one or more sports. Each sport can have its own batch.
⬆️ Import Students
📋 Required CSV/Excel format:
Name, RollNo, Sport, Batch, Age, DOB, Parent, Contact, Contact2, School, JoinDate
• Name is required. All other columns optional and can be in any order.
• DOB & JoinDate format: DD/MM/YYYY
• First row treated as header and skipped.
⬆️ Import Attendance
📋 Required CSV/Excel format:
Name, RollNo, Sport, Date, Status
• Name or RollNo is required to identify the student.
• Date format: DD/MM/YYYY (e.g. 15/06/2025)
• Status: P for Present, A for Absent
• First row treated as header and skipped.
⬆️ Import Fees
📋 Required CSV/Excel format:
Name, RollNo, Sport, Month, Status, Amount, Method, PaidDate
• Name or RollNo required to identify the student.
• Month format: YYYY-MM (e.g. 2025-06)
• Status: paid or unpaid
• Amount: number (e.g. 1500). Optional.
• Method: Cash / UPI / Card etc. Optional.
• PaidDate: DD/MM/YYYY. Optional.
• First row treated as header and skipped.
